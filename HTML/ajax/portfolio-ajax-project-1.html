Event Tube

Project Info:

We built this project to create a website that will pull a specific event from Eventbrite and with that information pull a video from a YouTube api. We were able to do this by fetching the data from the Eventbrite api and logging it to the YouTube search query. After the YouTube api populates with the Eventbrite data it will search YouTube for a video that is in relation to the specific event name. After it searches it will render this video onto an iframe on the page.The technologies used in this project were bootstrap.Some of the challenges we originally faced were getting the correct information to fetch from the Eventbrite api, fixing the responsiveness, various CSS and HTML issues, and MULTIPLE YouTube api key problems
